**目的 使用插件好处**

减少手动维护yapi接口文档的时间成本，

通过idea插件快速提交接口文档到yapi；

保证接口文档的准确性，

方便后台、前端、测试等查看接口文档；

反向规范代码注释，保持代码整洁；

保持公司统一风格的代码注释；
